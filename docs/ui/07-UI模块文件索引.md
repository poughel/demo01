## 六、文件命名规范

### 6.1 Vue 组件

```
✅ 推荐
ChartCard.vue       # PascalCase，清晰明确
StandardTable.vue
MiniArea.vue

❌ 避免
chartCard.vue       # camelCase
chart-card.vue      # kebab-case
chartcard.vue       # 全小写
```

### 6.2 JavaScript 文件

```
✅ 推荐
routerUtil.js       # camelCase
authorityUtils.js
themeUtil.js

❌ 避免
RouterUtil.js       # PascalCase
router_util.js      # snake_case

### 6.3 目录命名

```
✅ 推荐
colonyManage/       # camelCase
serviceManage/
hostManage/

colony-manage/      # kebab-case
