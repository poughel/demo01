# 告警数据访问层详解 - Infrastructure 层完整分析

## 一、数据访问层概述

### 1.1 模块定位

告警数据访问层是 DataSophon 告警管理系统的持久化组件，负责告警相关数据的增删改查操作。基于 MyBatis 和 MyBatis-Plus 框架，提供简洁高效的数据库访问接口。

**模块路径**: `datasophon-infrastructure/src/main/java/com/datasophon/dao/mapper/`

### 1.2 核心 Mapper 接口

```
告警数据访问模块
├── AlertGroupMapper                     # 告警组 Mapper
├── ClusterAlertQuotaMapper              # 告警指标 Mapper
├── ClusterAlertRuleMapper               # 告警规则 Mapper
├── ClusterAlertHistoryMapper            # 告警历史 Mapper
└── ClusterAlertGroupMapMapper           # 集群告警组映射 Mapper
```
